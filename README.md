Code for the Flatland Barriers experiments: learning and planning under changing environments.

Start with barriers.py (see comments on how to run at the top of that file): python3.12 barriers.py -h 

Please see the paper:

"When Remembering and Planning are Worth it: On Navigation Strategies under Change", O. Madani, B. Burns, R. Eghbali, and T. Dean. In Biologically Inspired Cognitive Architectures (BICA), 2025 (and Arxiv, 2026).

Specific requirements: pygame and gymnasium.
